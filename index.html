Smart Attendance is a modern employee attendance management system designed for businesses to track workforce attendance efficiently. Employees can sign up using their WhatsApp number and receive OTP verification for secure login. After creating their profile, users can mark attendance using real-time location verification. Key Features: - WhatsApp OTP Login - Employee Profile Management - Location-based Attendance - Secure Authentication System - Simple and Easy-to-use Interface Location is only used at the time of attendance marking and is not tracked continuously. This system is designed for companies, offices, and organizations to digitize attendance and improve workforce management.
