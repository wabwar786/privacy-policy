Privacy Policy

Smart Attendance App is designed for employee attendance tracking using WhatsApp OTP and location-based check-in.

Information We Collect

  • WhatsApp phone number for login
  • User name and profile details
  • Location (only during attendance marking)

How We Use Data

Data is used only for authentication and attendance tracking.

Location Usage

Location is used only at the time of attendance and is not tracked in background.

Data Sharing

We do not sell or share user data with third parties.
